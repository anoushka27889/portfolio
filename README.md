# Anoushka Garg Portfolio

A modern, performant portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Content**: MDX for case studies

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm start
```

## Project Structure

```
portfolio/
├── app/              # Next.js app directory (routes & pages)
├── components/       # Reusable React components
├── content/          # MDX content for projects
├── lib/              # Utility functions
└── public/           # Static assets (images, videos)
```

## Features

- Light/dark theme with localStorage persistence
- Optimized images with next/image
- Responsive design
- SEO optimized
- Accessibility focused
- Fast performance with static generation
